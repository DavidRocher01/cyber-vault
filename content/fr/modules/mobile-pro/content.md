# Sécuriser son téléphone professionnel

Votre smartphone est une cible de premier plan. Il contient vos emails, vos applications métier, vos authentifications MFA — souvent sans les protections d'un poste de travail classique.

---

## Les réglages de base indispensables

**Verrouillage d'écran fort**
- Utilisez un code PIN à 6+ chiffres ou biométrie (empreinte, visage)
- Activez le verrouillage automatique après 1-2 minutes d'inactivité
- Évitez les schémas de déverrouillage — trop prévisibles et visibles

**Mises à jour activées**
- Activez les mises à jour automatiques du système et des applications
- Les mises à jour corrigent des failles de sécurité critiques

**Chiffrement activé**
- Sur Android et iOS modernes, le chiffrement est activé par défaut si vous avez un code d'accès — vérifiez dans les paramètres de sécurité

---

## Les bonnes habitudes

**Applications officielles uniquement**
- Téléchargez uniquement depuis l'App Store ou le Play Store
- Méfiez-vous des applications tierces proposées via un lien ou un email

**Permissions raisonnables**
- Une application de lampe de poche n'a pas besoin d'accéder à vos contacts
- Vérifiez régulièrement quelles applications ont accès à votre micro, caméra, localisation

**Bluetooth et Wi-Fi**
- Désactivez le Bluetooth quand vous ne l'utilisez pas
- Évitez les Wi-Fi publics sans VPN (voir module dédié)

---

## En cas de perte ou vol

**Agissez dans les 30 premières minutes :**

1. **Localisez** votre appareil via "Trouver mon iPhone" ou "Find My Device" (Android)
2. **Verrouillez à distance** via votre compte Apple/Google
3. **Effacez à distance** si vous pensez que l'appareil est entre de mauvaises mains
4. **Signalez** à votre IT immédiatement — surtout si c'est un appareil professionnel

**Avant tout incident :** Activez ces fonctionnalités maintenant, pas après.

---

## BYOD — votre téléphone personnel pour le travail

Si vous utilisez votre téléphone personnel pour des activités professionnelles, les mêmes règles s'appliquent. Votre IT peut vous demander d'installer un profil de gestion (MDM) — c'est normal et légitime.

---

## Les SMS professionnels et l'ingénierie sociale

Les attaquants ciblent aussi les SMS (smishing) et les messages WhatsApp. Appliquez les mêmes réflexes qu'avec les emails : vérifiez l'expéditeur, ne cliquez pas sur les liens inattendus.
