# Checklist Vérification Naming — INPI / EUIPO / Domaine

> **Objectif :** Sécuriser le choix d'un nom de marque (ou confirmer le maintien de "CyberScan")
> **Temps requis :** 30 min pour pré-screening / 2h pour vérification approfondie
> **Coût :** 0€ pour vérif autonome / ~50-200€ pour recherche d'antériorité INPI complète par cabinet

---

## ⚠️ Avertissement préalable

**Ce document est un GUIDE de pré-screening, PAS un avis juridique.**

Pour tout dépôt définitif de marque ou en cas de doute :
- Consulter un **conseil en propriété industrielle (CPI)** agréé INPI
- Ou un **avocat spécialisé en droit des marques**
- Coût indicatif : 300-800€ pour un dépôt complet (3 classes en France)
- Économie potentielle : éviter un litige coûteux ou un dépôt refusé

---

## Étape 1 — Présélection des noms (autonome, gratuit)

### Critères d'un bon nom de marque

✅ **Distinctif** : pas un terme descriptif du produit ("cyberscan", "auditsécu" = REFUSÉS)
✅ **Court** : 5-10 lettres, mémorisable
✅ **Prononçable** au téléphone en français (test à faire à voix haute)
✅ **Disponible** en .fr ET .com (idéalement .io aussi)
✅ **Non associé** à une connotation négative ou autre secteur
✅ **Évolutif** : ne bloque pas une extension future de l'activité

### Filtre rapide (5 min par nom)

Pour chaque candidat, vérifier dans cet ordre :

1. **Google "nom_candidat"** → si page 1 vide de gros acteurs, continuer
2. **Google "nom_candidat cybersécurité"** → vérifier absence de concurrent direct
3. **Google "nom_candidat company"** ou "...société" → vérifier absence d'entreprise étrangère
4. **LinkedIn search** "nom_candidat" → vérifier absence d'entreprise ou consultant

Si les 4 voyants sont au vert, passer à l'étape 2.

## Étape 2 — Vérification INPI (France)

### URL : https://base-marques.inpi.fr

### Procédure

1. Aller sur **base-marques.inpi.fr**
2. Cliquer sur "Recherche avancée"
3. Champ "Marque" : entrer le nom candidat
4. Cocher "Marques en vigueur"
5. **Classes à vérifier en priorité** pour ton activité :

| Classe | Description | Pertinence |
|--------|-------------|------------|
| **9** | Logiciels, applications, programmes informatiques | 🔥 Critique |
| **35** | Conseil aux entreprises, gestion d'affaires | Forte |
| **38** | Télécommunications | Moyenne |
| **41** | Formation, éducation | Forte (si tu vends de la formation) |
| **42** | Services scientifiques et technologiques, conception/dev de logiciels, conseil cyber | 🔥 Critique |
| **45** | Services de sécurité, services juridiques (audit conformité RGPD) | 🔥 Critique |

### Interprétation des résultats

- ✅ **0 résultat sur les classes 9, 42, 45** → Très bon signe
- ⚠️ **Résultat dans une classe non concurrente** (ex: cosmétiques, alimentaire) → OK généralement, mais consulter un CPI pour confirmer
- ❌ **Résultat actif sur classes 9, 42 ou 45** → ABANDONNER ce nom
- ❌ **Marque "déchue" ou "expirée"** → Attention, le titulaire peut redéposer

### Recherche par similarité (avancée)

L'INPI examine aussi les noms **phonétiquement proches**. Faire 3-5 variations :
- Suppression/ajout d'une lettre (ex: "Sentinex" → "Sentinx", "Sentinexx")
- Variation phonétique (ex: "Klyra" → "Clyra", "Klira")
- Singularité/pluralité

## Étape 3 — Vérification EUIPO (Europe)

### URL : https://www.tmdn.org/tmview/

### Procédure

1. Aller sur **TMview** (la base européenne consolidée)
2. Entrer le nom dans "Trade mark name"
3. Filtrer par "EUIPO" et "France" + autres pays cibles (Belgique, Suisse, Luxembourg, Allemagne)
4. Filtrer par statut "En vigueur"

### Pourquoi c'est important

Une marque européenne (EUTM) couvre **les 27 pays de l'UE** automatiquement. Si quelqu'un a déposé "Sentinex" comme EUTM, tu ne peux pas l'utiliser en France même si rien n'est déposé à l'INPI.

### Procédure express alternative

URL : **https://euipo.europa.eu/eSearch/** → recherche directe sur les marques européennes.

## Étape 4 — Vérification noms de domaine

### Procédure

Pour chaque nom candidat, vérifier la disponibilité de :

| Extension | Priorité | Pourquoi |
|-----------|----------|----------|
| **.fr** | 🔥 Critique | Cible française, SEO local, crédibilité |
| **.com** | 🔥 Critique | International, défense de marque |
| **.io** | Forte | Perçu "tech" par les développeurs |
| **.eu** | Moyenne | Défense de marque |
| **.net, .org** | Faible | Défense de marque uniquement |

### Outils de vérification

1. **Whois rapide** : https://www.whois.com/whois/ (gratuit, instantané)
2. **Achat groupé** : Gandi.net, OVH, Namecheap (interfaces de recherche multi-extension)
3. **Domaines anciens** : https://www.expireddomains.net (parfois récupérable)

### Pièges à éviter

❌ **Acheter UNIQUEMENT le .fr** : tu seras vulnérable à un cybersquatting du .com
❌ **Acheter via revendeur cheap** : préférer Gandi/OVH pour la sécurité DNS
❌ **Domaines "look-alike"** (avec tirets, chiffres) : signal faible perçu par prospects

### Coût indicatif

- .fr : 8-12€/an
- .com : 12-15€/an
- .io : 35-60€/an
- **Total pack défensif (.fr + .com + .io) : ~70-90€/an**

## Étape 5 — Vérification réseaux sociaux (cohérence brand)

### Procédure

Vérifier la disponibilité du handle sur :

- **LinkedIn** : Company Page (search dans LinkedIn)
- **Twitter/X** : https://x.com/[nom]
- **Instagram** : https://instagram.com/[nom] (moins critique pour B2B)
- **YouTube** : https://youtube.com/@[nom] (si projet de chaîne)
- **GitHub** : https://github.com/[nom] (pour la crédibilité tech, oui c'est important)

### Stratégie

✅ Privilégier un nom où **tous les handles principaux** sont libres
✅ Si un handle est pris par un compte inactif (>2 ans), possible de le réclamer auprès de la plateforme

## Étape 6 — Vérification "smell test" final

Avant le dépôt définitif, 3 tests d'utilisateur :

### Test 1 — Le téléphone (5 personnes)

"Mon entreprise s'appelle **[nom]**, comment tu épelles ?"

→ Si >50% se trompent : nom trop ambigu, à reconsidérer.

### Test 2 — La mémorisation (5 personnes)

Dire le nom une fois. Discuter de tout autre chose pendant 5 min.
Demander : "Tu te souviens du nom de ma boîte ?"

→ Si >50% l'oublient : pas assez mémorable.

### Test 3 — L'association négative

Google Images "[nom]" + Google "[nom]" en mode incognito.

→ Si premières images/résultats inappropriés : à éviter.

## Étape 7 — Dépôt (quand tu es sûr)

### Option A — Dépôt autonome à l'INPI

URL : **https://www.inpi.fr/services-et-prestations/marque**

- **Coût** : 190€ pour 1 classe, +40€ par classe supplémentaire
- **Durée** : ~6 mois (avec phase d'opposition de 2 mois)
- **Difficulté** : moyenne, formulaire en ligne accessible
- **Risque** : 30% des dépôts autonomes ont des erreurs de classes ou de description

### Option B — Dépôt avec CPI / avocat spécialisé

- **Coût** : 400-900€ + taxes INPI
- **Avantage** : recherche d'antériorité approfondie + rédaction des classes optimale
- **Recommandé si** : tu vises >100k€ de CA sur la marque dans les 3 ans

### Option C — Dépôt européen (EUTM)

URL : **https://euipo.europa.eu**

- **Coût** : 850€ pour 1 classe (EUTM), couvre 27 pays UE
- **Recommandé si** : ambition européenne dès le départ
