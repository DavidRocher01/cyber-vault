Offres audit


1. Solution "FLASH" (L'entrée de gamme)

Cible : Indépendants, petits sites vitrines ou blogs.
C'est un diagnostic rapide, presque entièrement automatisé, qui sert de "prise de conscience" pour le client.

    Contenu technique :

        Scan de vulnérabilités externes (OWASP ZAP / Nuclei).

        Analyse de la configuration SSL/TLS (SSL Labs).

        Vérification des headers HTTP de base.

        Scan de ports (Nmap) pour vérifier l'exposition du serveur.

    Livrable : Un rapport synthétique de 3-5 pages avec un score de sécurité (A à F).

    Temps de travail : ~2 à 3 heures.

    Prix conseillé : 450 € à 600 € HT.

2. Solution "APP-CHECK" (Spécialité Angular/FastAPI)

Cible : Startups, SaaS, applications métiers avec espace client.
C'est ici que tu valorises ton expertise de développeur. Tu ne regardes pas seulement la "porte", mais aussi les "fondations".

    Contenu technique (Flash +) :

        Audit de code statique (SAST) : Analyse de ton dépôt Git.

        Scan de dépendances (SCA) : Liste des bibliothèques vulnérables dans package.json et requirements.txt.

        Recherche de secrets : Détection de clés d'API ou mots de passe "en dur" dans le code.

        Audit de la base de données : Vérification des droits d'accès PostgreSQL.

    Livrable : Rapport détaillé de 10-15 pages avec plan d'action priorisé.

    Temps de travail : ~1 à 2 jours.

    Prix conseillé : 1 200 € à 1 800 € HT.

3. Solution "PENTEST LÉGER" (Le haut de gamme)

Cible : E-commerce, sites manipulant des données sensibles (santé, finance).
Ici, tu ne te contentes pas de scanner, tu essaies de "rentrer" comme un vrai hacker.

    Contenu technique (App-Check +) :

        Tests d'intrusion manuels : Tentative d'injection SQL, bypass d'authentification, manipulation de jetons JWT.

        Test de logique métier : Est-ce qu'un utilisateur A peut voir les données de l'utilisateur B en changeant un ID dans l'URL ?

        Audit de configuration Cloud : Vérification des droits sur AWS/Azure/OVH.

    Livrable : Rapport complet incluant les preuves d'exploitation (screenshots) et recommandations de code précises.

    Temps de travail : ~3 à 5 jours.

    Prix conseillé : 3 000 € à 5 500 € HT.
